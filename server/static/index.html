Environment Spec

Clean state, typed models, reproducible rewards.

The environment implements the OpenEnv interface end-to-end: typed Action and Observation models, explicit reset() / step() / state() endpoints, and reward shaping that penalises inefficiency while preserving partial progress signal.

  • Classify — per-email exact match, aggregated over five dispatches.
  • Prioritise — Kendall-tau over the urgency ordering.
  • Reply — five-criterion rubric on the dispute response.
